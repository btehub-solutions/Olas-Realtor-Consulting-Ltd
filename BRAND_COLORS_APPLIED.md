## Brand Color Palette

### Primary Colors
- **Deep Emerald Green**: `#006B3C` - Main brand color
  - Used for: Navigation elements, headings, primary text, form focus states
  - CSS Variable: `--primary-green: #006B3C`

- **Deep Maroon**: `#580F0F` - Secondary accent color
  - Used for: CTAs, hover states, highlights, badges, prices

- **Off-White**: `#FAFAFA` - Alternate section backgrounds
  - CSS Variable: `--off-white: #FAFAFA`

## Nigerian-Inspired Design Philosophy

The color palette reflects Nigeria's rich heritage:
- **Deep Emerald Green** represents Nigeria's lush landscapes, growth, and prosperity
- **Deep Maroon** symbolizes strength, passion, and the earth's richness
- The gradient combination creates a modern, sophisticated look while maintaining cultural authenticity

## Accessibility Considerations

- All color combinations meet WCAG 2.1 AA standards for contrast
- High contrast mode support included with adjusted colors:
  - Primary Green: `#005030`
  - Maroon: `#450D0D`
- Focus states use 3px solid outlines with the primary green color
- Reduced motion support for users with vestibular disorders

## Browser Compatibility

- CSS variables are supported in all modern browsers
- Fallback colors are provided where necessary
- Theme-color meta tag updated for mobile browser chrome
